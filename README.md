# WM 2026 Score Predictor

## Ziel
Prediction Model für ein WM Tippspiel.

Vorhersage des exakten Spielstands (z. B. 2:1).

## Geplante Features
- Kaderwert (Transfermarkt)
- Teamform (letzte 5 Spiele)
- Spielerform (letzte 10 Spiele)
- Elo Rating
